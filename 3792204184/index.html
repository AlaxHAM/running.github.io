pyquery使用

模块安装

​ pyquery 的出现使得对网页的元素的提取更加方便，毕竟如果全都是使用正则表达式，这样编写的话内容量会很大，并且很容易出现匹配结果不理想，pyquery 可以直接对 html 的 DOM节点结构进行解析，并且使用 DOM 节点的属性来进行内容个提取

python

初始化

​ 在对 html 文档进行解析前，需要将内容初始化为成 pyquery 对象，包括可以直接是 html 的字符串进行初始化，也可以 直接使用 url 进行初始化，或是将 *html 的本地文件 *作为参数初始化

python

​ print 函数会将 html 字符串中的所有 li 元素打印出来，同过 type 函数得到 <class ‘pyquery.pyquery.PyQuery’> 对象，当然这并没有将标签内的文本提取出来

python

​ 通过url参数，pyquery会先请求这个url获取到响应对象，从中得到整个页面的html结构文档，并进行初始化得到 pyquery 对象

python

​ 通过 filename 参数可以指定本地的 html 文件，这样也可以进行初始化得到 pyquery 对象

基本CSS选择器

​

* 选取所有节点
#container 选取 id 名为 container 的节点
.container 选取 class 名为 container 的节点
li a 选取所有 li 标签下的 a 标签（包括子和后代）
ul+p 选取 ul 后面是 p，且p与ul是同级（兄弟）的节点
ul>p 选取父级是ul的所有p节点
p~ul 选择 p 标签后面出现的所有 ul 标签节点，不管是子节点，兄弟节点
a[title] 选取所有属性有title的 a 元素
a[href='http://ah'] 选取href属性值是”http://ah"的所有 a 元素
a[href*='ham'] 选取href属性中包含’ham’字符的 a 元素
a[href^='http'] 选取href属性中以’http’字符开头的 a 元素
a[href$='.jpg'] 选取href属性中以’，jpg’字符结尾的 a 元素
input[type=radio]:check 选取所有被点击的单选按钮的 input 元素
div:not(#wapper) 选择所有 id 不为 wapper 的 div 元素
li:nth-child(3) 选择从父元素下索引为3的 li 元素
li:nth-child(2n) 选择从父元素下索引为偶数的 li 元素

遍历节点

​ 通常使用选择器获取到的大多都是一个集合，其中包含了全部满足选择器的所以节点，所以通过循环遍历每一个节点，在使用 text 方法就可以得到标签中的文本内容

python

​ itmes 方法会返回一个生成器对象，每一次遍历都会从中取出一个元素

子节点

​ 对于子节点的查找使用的是 find 方法，传入的参数就是使用的 CSS选择器，比如在上个代码中是获取到 id 为 container 下 li 标签中 a 标签，但是如果在之后的操作中任然有要提取的内容，且同样是在 li 标签下，此时就无法在使用之前的 pyquery 对象，只能再次重新写 CSS选择器

​ 所以由此可以直接将 li 这一层获取得到一个 pyquery 的对象，通过使用 find 方法，可以多次使用这个对象

python

​ 这里的 a_items， p_items 分别是包含了 li 下的所有 a 和 p 标签的一个 pyquery 对象，通过遍历后就可以获取各自对应的文本，所以 find 方法查找的范围是给出范围下的 所有子孙节点，对于 直接查找子节 点，可以使用 children 方法获取

python

​ 当然使用 children 查找子节点也可以使用 CSS选择器 来指定具体的某一个子节点

python

父节点

​ 对应子节点，父节点也有两种，一种是直接的父级节点，也就是当前元素节点直接的上一层而不是继续往上，另一种就是继续往上找祖先节点，两种方法分别是 parent 和 parents

​ parent 方法返回的结果是直接的父级节点所以只有一个，而是用 parents 方法会把所有包含自己的上层节点全部返回，所以使用 parents 方法想要具体的一个父级节点，同样可以使用CSS选择器去指定

兄弟节点

​ 很明显这是用来查找相邻节点的，使用的方法是 siblings ，会将所有的相邻兄弟节点全部返回，如果要获取特定的一个节点，一样使用 CSS选择器 即可

属性与文本

属性值获取

​ 在得到节点后，通过遍历可以获取到 pyquery 中每一个元素节点，接下来就是内容的提取，属性的获取可以通过 attr 方法指定属性名

python

​ attr 方法还可以直接用 ‘ .[属性名] ‘ 的方式来得到该属性的值

python

​ 需要注意的是这里的节点只有一个，当有多个节点的情况下，直接使用 attr 来获取属性，只会需要第一个元素节点的该属性的值，所以当对多个节点获取属性值，依然需要使用循环遍历的方式

python

文本内容获取

​ 之前的文本提取中已经使用了一个方法 text，那还有另一个方法就是 html，两者都可以获取到节点的内容，区别在于：

​ text 方法会忽略节点内 html 的语法，返回纯文字的内容；

​ html 方法则是会将当前节点内的子标签文本内容和html文本一并返回

节点操作

样式修改

​ 对于动态修改一个节点的属性，进行属性的添加或删除，有两个方法：addClass 和 removeClass ，只需要将属性名传入即可，这两个方法只能是对节点的样式属性进行操作，有时候可能会见到 add_class 和 remove_class 的写法，实现的结果是一样的，addClass 相当于是 add_class 的别名

python

属性添加

​ 之前的 attr 方法，有两种操作方式，在不使用参数或只使用一个参数 ‘属性名’ 的情况下，attr 能够获取到这个属性的值，但是如果添加第二个参数，那么 attr 方法会将指定的属性名加入节点，并将 第二个参数作为该属性的值，如果 attr 的这个属性在节点已存在，并且给了第二个参数，那么第二个参数的值会覆盖节点原来属性的值

python

文本内容修改

​ 与 attr 方法类似，text 和 html 方法可以用来获取节点中的纯文本或是整个节点字符串，text 和 html 方法中如果插入字符串参数，可以对原有内容进行修改

python

删除节点

​ 有时候在得到节点后使用 text 方法会得到所有节点下的所有文本内容，但有时可能并不是却不都需要的，所以可以通过删除不用提取的子节点，使用 remove 方法即可

python

​ empty 方法同样可以删除，但是它执行的是清空的操作，会将当前调用的父元素下所有子元素删除

添加节点

​ append 方法可以在父元素中将子元素进行添加，还有 append_to 方法则是将子元素追加到父元素中，伪代码就是： 父元素.append(子元素)， 子元素.append_to(父元素)

python

​ append 默认都是在最后进行子元素的添加，所以该有 prepend 的方法来实现前置添加的效果，将子元素加入到第一个位置，和 append 对应也有 prepend_to 方法

​ 除了在最前和最后这种针对父子节点之间的操作，兄弟节点也有前后进行元素添加的方法，如： after，insert_after，before，insert_before，带有 ‘_’ 的都是代表要插入的元素在指定的元素前或后进行添加

伪类选择器

​ 在 CSS 中可以使用伪类来为同一种标签指定不同的状态，从而使用不同的样式，比如鼠标悬浮链接 ‘ a:hover ‘，定义这样的伪类可以让鼠标移动到a标签上，去自定制此时 a 标签的一个样式，而其他的a标签则会是使用默认的样式

​ CSS3 中提供节点位置的伪类选择器：

​ li:first-child：获取第一个 li 节点

​ li:last-child：获取最后一个 li 节点

​ li:nth-child(2)：获取第二个 li 节点

​ li:gt(2)：获取第二个 之后的所有 li 节点，gt 是大于的意思，2是下标，在 html 的节点中下标从 1 开始

​ li:nth-child(2n)：获取偶数位的 li 节点

​ li:contains(name)：获取文本内容包含 ‘name’ 的 li 节点
